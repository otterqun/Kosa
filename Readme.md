# 🇲🇾 Kosa - Klon Wordle Bahasa Melayu

Satu permainan teka perkataan interaktif dalam Bahasa Melayu yang padat dengan ciri-ciri tambahan, dibina untuk dimainkan terus di pelayar web anda.

Projek ini sangat diinspirasikan daripada permainan fenomena global, **[Wordle](https://www.nytimes.com/games/wordle/index.html)** ciptaan Josh Wardle. Walau bagaimanapun, aplikasi ini dibangunkan sepenuhnya dari bawah (scratch) sebagai **projek individu** untuk mengasah kemahiran pembangunan bahagian hadapan (Frontend Web Development) dan menyuntik elemen pendidikan Bahasa Melayu.

## ✨ Ciri-ciri Utama (Features)

Selain daripada mekanik asal Wordle, projek ini merangkumi pelbagai penambahan unik:
* **Mod Kepelbagaian Huruf:** Pemain bebas memilih untuk mencabar minda dengan perkataan bersaiz **4 Huruf**, **5 Huruf**, atau **6 Huruf**.
* **Integrasi Kamus (Pendidikan):** Setiap kali perlawanan tamat (menang atau kalah), makna rasmi bagi perkataan tersebut akan dipaparkan.
* **Sistem Tema Dinamik (Multi-themes):** Pemain boleh menukar rupa bentuk visual permainan secara masa nyata (real-time). Tema yang disertakan:
  - 🌙 **Gelap (Dark)** - Mod lalai yang moden dan elegan.
  - ☀️ **Terang (Light)** - Selesa untuk mata di tempat terang.
  - 🌴 **Village** - Estetika warna alam, kayu, dan daun.
  - 💻 **Windows XP** - Tema nostalgik tahun 2001 dengan butang berbayang tebal.
  - 📄 **Raw HTML** - Paparan asas web tanpa gaya untuk rasa retro/minimalis.
* **Pautan Rujukan Segera:** Dropdown pantas untuk merujuk perkataan di Pusat Rujukan Persuratan Melayu (PRPM) DBP.

## 🛠️ Teknologi yang Digunakan (Tech Stack)

Aplikasi ini dibina dengan seni bina Single Page Application (SPA) dan Client-Side Rendering (CSR) menggunakan teknologi moden berikut:

* **[Vue.js 3 (Composition API)](https://vuejs.org/)** - Kerangka kerja JavaScript utama untuk membina komponen antaramuka.
* **[Pinia](https://pinia.vuejs.org/)** - Pengurusan *state* (State Management) yang ringan untuk menyimpan logik permainan, pangkalan data perkataan, dan memori tema.
* **[Tailwind CSS](https://tailwindcss.com/)** - Utiliti CSS untuk susun atur yang responsif (*mobile-friendly*) dan penggayaan (*styling*) yang pantas.
* **[Vite](https://vitejs.dev/)** - Alat pembinaan (Build Tool) yang sangat pantas untuk kompilasi kod secara tempatan.
